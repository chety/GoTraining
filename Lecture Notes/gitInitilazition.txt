If you want to create a repo from scratch here is the milestones.

1)Go to github.com or whatever your favourite git provider and create a repo. Do not add any gitignore ...vs files.
2) In your locale open Git bash and head to your project directory.
3) "Git init" -> This initialize your empty git repo
4) From your favourite ide add and .gitignore and README.md file.
5) In Gitbash  write "git add -all"(Add all new files) or "git add .gitignore" (add specific file)
6) "git commit -m 'first commit'". Before committing you can see your git status via "git status"
7) This command must be run just once. "git remote add origin  https://github.com/username/reponame.git"
This command ensures that your local repo connects to the right github repo.
8)"git push -u origin master".

Thats all.
